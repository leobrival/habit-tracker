# Checker API Documentation

Welcome to the Checker API documentation. This API allows you to integrate habit tracking functionality into your applications, including Raycast extensions, CLI tools, and custom integrations.

## Base URL

```
https://api.checker.app/v1
```

## Quick Start

1. [Create an account](/getting-started/) and get your API key
2. [Authenticate](/authentication/) your requests using your API key
3. [Start tracking habits](/api-reference/boards/) with the Boards API

## Features

- **Habit Boards**: Create and manage unlimited habit tracking boards
- **Check-ins**: Record daily habit completions with optional quantities
- **Statistics**: Access streaks, completion rates, and heatmap data
- **Quick Actions**: Optimized endpoints for Raycast and CLI tools

## Rate Limits

| Plan | Requests/minute | Requests/day |

| Pro | 300 | 10,000 |
